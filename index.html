Song Finder

Song Finder is a dynamic and interactive web application that allows users to discover songs based on lyrics or obtain lyrics by providing the artist and song name. Built using Python, Flask, Genius API, HTML and CSS.

Image Filters

This program replicates core features of image editing softwares that allow you to add image filters. This program offers functionalities such as greyscale conversion, compression, mirroring, color inversion, and a stripe patterns.
